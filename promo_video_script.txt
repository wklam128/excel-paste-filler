==========================================================
RECORDING TIPS
==========================================================
- Use Windows Game Bar (Win + G) or OBS Studio to record
- Set browser zoom to 100% before recording
- Record at 1920x1080 for best quality
- Use a quiet room if recording voiceover with microphone
- Alternatively use free text-to-speech tools:
    - ElevenLabs (elevenlabs.io) — very natural voice
    - Microsoft Azure TTS — free tier available
    - Murf.ai — good for professional voiceover
- Recommended video editor: DaVinci Resolve (free)
- Export as MP4, 1080p, H.264
- Upload to YouTube as Public or Unlisted
- Paste the YouTube URL into Chrome Web Store listing
