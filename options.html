Welcome to Excel Paste Filler!

Paste Excel cells into any web form and auto-fill adjacent fields in order. Select any text on the page to instantly translate it. By default the addon is active on all pages — restrict it via the Active on These URLs list below, or click the extension icon on any page to quickly add or remove that site.
