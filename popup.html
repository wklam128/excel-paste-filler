How to use
① Paste Excel data into fields
  1. 1Select cells in Excel → Ctrl+C
  2. 2Click the first field to fill
  3. 3Press Ctrl+V — fills automatically
② Copy table from page to Excel
  1. 1Right-click inside any table
  2. 2Choose "Copy Table to Clipboard"
  3. 3Paste into Excel — layout preserved
